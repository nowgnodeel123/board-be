# 📋 게시판 프로젝트 (Board Project)

이 프로젝트는 Spring Boot와 React를 사용하여 기본적인 CRUD 기능을 갖춘 게시판을 구현하는 것을 목표로 합니다.

## ✨ 주요 기능 (Features)

### 📄 게시글 (Post)
- [ ] 게시글 작성 (Create)
- [ ] 게시글 목록 및 상세 조회 (Read)
- [ ] 게시글 수정 (Update)
- [ ] 게시글 삭제 (Delete)

### 💬 댓글 (Comment)
- [ ] 댓글 작성 (Create)
- [ ] 댓글 조회 (Read)
- [ ] 댓글 수정 (Update)
- [ ] 댓글 삭제 (Delete)
